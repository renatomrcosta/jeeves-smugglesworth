Features

Subtext for header

Merge

Requests Jeeves for a spot in the merging queue.

Done

Signals the end of your merge. However, it only works if you are the one in the merge queue at the moment

Kick

Kicks whoever is in the queue at the moment. Good for those pesky git and runners, but use it wisely!

Status

Shows the status of the merge queue for the current channel

Help

Shows a friendly message!
