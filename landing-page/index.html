Jeeves, your friendly merge flow butler!

Jeeves will watch your slack channels and keep track of your merging procedure!!!

Find me on GitHub

Features

Fun stuff you can do with Jeeves

Merge

Requests Jeeves for a spot in the merging queue.

Done

Signals the end of your merge. However, it only works if you are the one in the merge queue at the moment

Kick

Kicks whoever is in the queue at the moment. Good for those pesky git and runners, but use it wisely!

Status

Shows the status of the merge queue for the current channel

Help

Shows a friendly message!

Testimonials

At our current project we are using Jeeves. It’s a big timesaver since we always know if somebody is merging with just a small command as @jeeves status in Slack. Because it also checks if nobody is merging when you do a request for a new merge we didn’t lose time with two persons merging at the same time.

Overall a great product with a high uptime. Keep up the good work!!!!
